Online Courses

Kursus web adalah program pelatihan yang ditujukan untuk mengajarkan keterampilan dalam membangun, merancang, dan mengelola situs web.

Dalam kursus web, peserta akan belajar mengenai berbagai teknologi web seperti HTML, CSS, JavaScript, dan PHP serta alat-alat pembuat situs web seperti WordPress, Drupal, atau Joomla. Peserta juga akan diajarkan tentang aspek-aspek desain web seperti user experience (UX), user interface (UI), dan responsif design.
